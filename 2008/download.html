Design Contest Data Set Downloads

This page contains links to download the data files. See the Data Description page for information on how to interpret the files and what they mean, and for example images.

Scalar Field Data

The scalar field data (density, temperature, and the eight chemical species abundancies are stored in files anmed multifield.XXXX.txt, where XXXX stands for the time index.

To provide a smaller data set that displays the entire simulation, a single XY slice through the data set near the center of the simulation volume for all times has been extracted into individual .txt files, which are grouped together into a single ZIP file named zslices.multifield.zip.

Vector Field Data

The scalar field data (density, temperature, and the eight chemical species abundancies are stored in files anmed velocity.XXXX.txt, where XXXX stands for the time index. The index matches the frame for the scalar field data at the same simulation time step.

Download

Data can be downloaded from following locations
  • SDSC Cloud : You could easily download all data by using wget as follows
    % wget -r -nH --no-parent --cut-dirs=2 https://cloud.sdsc.edu/v1/AUTH_sciviscontest/2008/Data
    % wget -r -nH --no-parent --cut-dirs=2 https://cloud.sdsc.edu/v1/AUTH_sciviscontest/2008/data_files
